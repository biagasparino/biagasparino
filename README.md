## 👩‍💻 About me

I'm a **Monitoring & Observability Analyst** with experience in large-scale corporate environments. Day to day, I ensure the visibility and health of critical infrastructures — from configuring metrics and alerts to incident analysis and ITSM workflow integration.

My focus is on building end-to-end observability: data collection, event correlation, actionable dashboards, and response automation. I'm currently moving into more senior SRE and Platform Engineering roles, with active studies in Kubernetes and Prometheus.

### 🔭 [observability-homelab](https://github.com/biagasparino/observability-homelab)
> Full monitoring stack with Prometheus, Grafana, Zabbix and Alertmanager running on Docker Compose

Local observability stack built to study and demonstrate real SRE practices:
- Metrics collection with **Prometheus** + exporters (node, blackbox)
- SLO/SLI dashboards on **Grafana**
- Alerts configured in **Alertmanager** with Slack notifications
- Automatic dashboard provisioning via `grafana.ini`
